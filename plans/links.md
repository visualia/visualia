# DSLinks picks — the 2019 designstem canon

The enduring gems hand-picked from the 539-link 2019 designstem corpus (deduped against the library above; the rest were dated, thin, or off-theme). The analytical reading of this corpus is in [topics.md](topics.md).

## Explorable explanations & reactive documents

- [Explorable Explanations (Nicky Case essay)](https://blog.ncase.me/explorable-explanations/) — foundational essay naming and dissecting the form
- [Explorable Explanations: 4 More Design Patterns](https://blog.ncase.me/explorable-explanations-4-more-design-patterns/) — reusable interaction patterns for reactive docs
- [Distill](https://distill.pub/) — the gold standard for interactive, rigorous explanation
- [Using AI to Augment Human Intelligence (Distill)](https://distill.pub/2017/aia/) — Carter & Nielsen manifesto-as-explorable on cognitive prosthetics
- [A Visual Introduction to Machine Learning (R2D3)](http://www.r2d3.us/visual-intro-to-machine-learning-part-1) — the scrollytelling ML piece everyone cites
- [A Neural Network Playground (TensorFlow)](http://playground.tensorflow.org) — direct-manipulation neural net in the browser
- [Parametric Press](https://parametric.press/issue-01/) — a whole publication of explorables on science + society

## Tools for thought / dynamic medium (classics)

- [Magic Paper (Michael Nielsen)](http://cognitivemedium.com/magic_paper) — sketch-driven interactive reasoning; core tools-for-thought reference
- [قلب (Ramsey Nasser)](https://nas.sr/%D9%82%D9%84%D8%A8/) — Arabic programming language probing how culture shapes code

## AI interfaces / creative AI on canvas

- [Chalktalk (Ken Perlin)](https://github.com/kenperlin/chalktalk) — draw-to-animate live presentation medium; seminal spatial interface
- [Fast Drawing for Everyone (Google AutoDraw)](https://www.blog.google/technology/ai/fast-drawing-everyone/) — early ML-assisted drawing

- [LOOPY](https://ncase.me/loopy/) — draw-and-simulate systems-thinking canvas

- [Domain Coloring for Complex Functions](https://observablehq.com/@rreusser/domain-coloring-for-complex-functions) — visualizing 4D complex maps
